## 🚀 ЧТОБЫ ЗАПУСТИТЬ ОНЛАЙН:

### 1️⃣ Включите GitHub Pages

1. Откройте: **https://github.com/artteth/canban/settings/pages**

2. В разделе **"Build and deployment"**:
   - **Source**: Deploy from a branch
   - **Branch**: online
   - **Folder**: / (root)

3. Нажмите **Save**

### 2️⃣ Подождите 1-2 минуты

GitHub Pages построит ваш сайт.

### 3️⃣ Ваш сайт доступен:

```
https://artteth.github.io/canban/
```
