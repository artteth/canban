# 📋 Kanban Board - Система управления задачами

## 🎯 Описание

Система управления задачами с интеграцией Telegram и поддержкой нескольких пользователей.

## 🚀 Быстрый старт (локальное тестирование)

### Вариант 1: Локальное тестирование (без настройки)

1. Откройте в браузере:
   ```bash
   open v2/select_user_local.html
   ```

2. Протестируйте:
   - Выбор пользователя
   - Создание задач
   - Переключение между колонками
   - Админскую панель

**Все данные сохраняются в localStorage браузера!**

### Вариант 2: Полноценная установка

См. подробную инструкцию: **[v2/SETUP_INSTRUCTIONS.md](v2/SETUP_INSTRUCTIONS.md)**

Кратко:
1. Создать Google Таблицу
2. Настроить Google Apps Script (cod_v2.gs)
3. Развернуть веб-приложение
4. Обновить API_URL в frontend файлах

## 🎯 Сценарии использования

### Сценарий 1: Обычный пользователь
1. Открывает `select_user_local.html`
2. Выбирает себя из списка
3. Создаёт задачу → назначает на себя
4. Перемещает по колонкам

### Сценарий 2: Администратор
1. Открывает `admin_local.html`
2. Переключается в режим "Доска"
3. Фильтрует по пользователю "Игорь"
4. Создаёт задачу → назначает на Игоря

### Сценарий 3: Telegram Bot
1. Пользователь пишет: `/add Купить молоко`
2. Бот добавляет задачу
3. Подтверждает сообщением

## 🐛 Решение проблем

### Задачи не загружаются:
- Проверьте API_URL
- Проверьте права доступа к Google Apps Script
- Откройте консоль браузера (F12)

### Пользователи не сохраняются:
- Проверьте наличие листа "Пользователи"
- Проверьте заголовки столбцов

### Ошибки CORS:
- Используйте JSONP режим
- Или разверните на том же домене
